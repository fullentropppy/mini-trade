1С: МиниТорговля: простая конфигурация для ведения учета торговли в небольшой организации.

## Функционал
- **Склад**: оприходование, перемещение между складами, списание товаров;
- **Торговля**: поступление, продажа товаров, установка цен товаров;
- **Отчетность**: склад и торговля;
- **Предприятие**: данные организации, настройки учета;
- **Система**: управление пользователями, регламентными и фоновыми заданиями, консоль кода и запросов и др.

## Установка
Требуется платформа не ниже версии `8.3.16`.

1. Скачать исходный код и распаковать `.zip` или клонировать репозиторий;
2. Сделать резервную копию базы данных перед загрузкой файлов конфигурации (если конфигурация-приемник не пустая);
3. В конфигураторе: `Конфигурация` > `Загрузить конфигурацию из файлов` > выбрать папку `src` загруженную на шаге **1**.
